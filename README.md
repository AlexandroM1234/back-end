# Expat Journal Backend

## About

- An api for a Instagram like web application

## Install

    $ git clone https://github.com/bw-expat-journal-1/back-end.git
    $ cd how-to
    $ npm install

## Start

    $ npm start

## TechStack

- Bycrypt
- Cloudinary
- Cors
- Express
- NodeJS
- JsonWebToken
- SQLite3
- PostgresSQL
- Nodemon
- Helmet
